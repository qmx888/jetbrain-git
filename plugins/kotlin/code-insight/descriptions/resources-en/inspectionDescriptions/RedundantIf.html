Reports boolean if expressions and statements which can be simplified to a single expression.

Example:


  fun test(): Boolean {
      if (foo()) {
         return true
      } else {
         return false
      }
  }

After the quick-fix is applied:


  fun test(): Boolean {
      return foo()
  }

The inspection also simplifies branches where one branch is a boolean constant:


  val filterModified = if (condition) {
      someBooleanExpression
  } else {
      false
  }

After the quick-fix is applied:


  val filterModified = condition && someBooleanExpression
