Reports value?.hashCode() ?: 0 and suggests replacing it with value.hashCode().

Since Kotlin 1.3, the standard library provides Any?.hashCode(), which returns 0 for null and delegates to the receiver's hashCode() otherwise.


fun hash(value: String?): Int {
    return value?.hashCode() ?: 0
}

After the quick-fix:


fun hash(value: String?): Int {
    return value.hashCode()
}
